# Quickstart: Testing the Route Package

To run the new automated tests for the route package:

1. Ensure you are in the project root directory.
2. Run the Go test command specifically targeting the route package:

```bash
task test
```

To view a detailed coverage report for the route package:

```bash
task test-cover
```

*Note: This utilizes the existing `Taskfile.yaml` set up in previous test implementations.*
